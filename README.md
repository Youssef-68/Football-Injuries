# Football-Injuries
This project analyzes 110K+ sports injuries (1995–2025). Using Power BI, I built dashboards to visualize average days/games missed, injury causes, and yearly trends.  Avg. days missed: 53.6  Avg. games missed: 7.2  Top causes: Muscle (15%), Hamstring (13%), Knee (12%) These insights can help clubs improve player management and reduce injury impact.
